# Goal: Understand Redis basics, install it, and perform simple operations.

## Topics:

1. What is Redis?
    - In-memory key-value store
    - Use cases: caching, sessions, queues, leaderboards

2. Installing Redis

    - Linux, macOS, Windows (WSL/Docker)

3. Redis CLI basics

    - SET, GET, DEL, EXISTS
    - TTL (EXPIRE)
    - KEYS * and SCAN

4. Data types basics

    - Strings, Lists, Sets, Hashes, Sorted Sets

5. Basic configuration

    - redis.conf
    - Changing port, persistence, password

Exercises:

- Install Redis on your machine
- Store and retrieve keys in CLI
- Experiment with TTL on keys
